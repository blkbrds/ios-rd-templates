This is iOS Team 's template used in our projects using Xcode Templates

## I. Getting Started
### Requirements
- Xcode: `14.0 or later`
- Rbenv: `1.2 or later`
- Ruby: `2.7.2 or later`
- Bundler: `2.1.4 or later`
